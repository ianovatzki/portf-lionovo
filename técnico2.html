Trabalho de Modelagem de sistemas

Metodologias de desenvolvimento

Nesse trabalho tivemos fazer uma apresentação sobre uma determinada metodologia de desenvolvimento. Escolhi a Kanban, junto com o meu grupo, pois é a mais fácil, além disso explicamos sua história, seu criador, e mostramos como ela poderia ser feita. Habilidades: H3 - H5 - H8

Trabalhos de Banco de dados

Lista - Modelo Lógico

Nessa atividade tive que resolver uma lista de exercícios utilizando o modelo lógico. Utilizei o brmodelos para fazer os exercícios, interligando relacionamentos e entidades. Habilidades: H1 - H2

Pesquisa normalização de dados

Nesse trabalho tive que explicar sobre a normalização de dados e para que servia. Utilizando o roteiro proposto, pesquisei e desenvolvi o roteiro com base em pesquisas, usando o site da microsoft como base. Habilidades: H3

Modelagem Sistema de Gestão de Consultório Médico

Nesse trabalho tivemos que criar um modelo conceitual e um modelo lógico para um sistema de gestão de um consultório médico, utilizando o brmodelos, criamos as entidades e seus respectivos relacionamentos.
