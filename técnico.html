Esta é a página do técnico, onde irei apresentar oito trabalhos que fizemos nesse trimestre.

Trabalhos de desenvolvimento de sistemas

Calculadora de horas

Nesse trabalho tivemos que fazer uma calculadora de horas, contando a hora inicial, o minuto inicial, a hora final e o minuto final. Achei esse trabalho útil para entender os conceitos do React e do Javascript, que nos permitiu entender um pouco mais a respeito. Além disso nos foi ensinado o vite, que nos fez aprender o conteúdo de maneira mais suave e sem muitos problemas. Habilidades: H3, H4, H5, H6, H9, H10

API - github

Nesse trabalho tivemos que programar uma API que buscava um usuário no github. Utilizamos o react como base para este projeto. Habilidades: H3 - H4 - H5 - H6 - H9 - H10

Trabalho API

Nesse trabalho tivemos que fazer uma apresentação sobre API, assim como os estilos de arquitetura, linguagens de consulta, suas ferramentas e verbos http e https. Habilidades: H3 - H4 - H5 - H6 - H9 - H10

Trabalhos de Modelagem de sistemas

Diagrama SA

Nesse trabalho tivemos que criar um UML para nossa SA, utilizando os casos de uso e caminhos adequados para se ter uma base de como faremos a SA sem complicações. Achei o trabalho muito útil, pois o UML pode ser bem confuso para a gente, e também porque nos ajudou a ter uma coordenação da trabalho da SA, nos dando uma ideia de como podemos fazer.

Gerência de Projeto para Implantação de um Sistema de CRM em Desenvolvimento

Nesse trabalho tivemos que criar nossa própria gerência de projeto na ferramenta Wrike, colocando desde o início até o fim do projeto com determinadas funções para a "equipe". Esse foi um trabalho puxado, pois tivemos que mudar tudo certinho para a equipe, tudo deveria estar nas datas e descrições certas, mas no final conseguimos acabar o trabalho de maneira correta.

Wireframe de alta fidelidade

Nesse trabalho tivemos que criar um wireframe de alta fidelidade no Figma para nossa SA. O projeto ainda está em andamento, porém já construímos sua interface web e mobile. faltando agora a interface web no mobile. Habilidades: H2 - H3 - H4 - H5

Regras de negócio(SA)

Nesse trabalho tivemos que criar nossas regras de negócio para nossa SA, utilizando o google planilhas colocamos 15 regras de negócio, o projeto pode ainda ser complementado com ainda mais regras. Habilidades: H2 - H3 - H4 - H8

Atividade de Implantação e Manutenção de Sistemas

Cenário 1 Atividade 1

Nesse trabalho tivemos que propor um modelo de instalação para uma empresa, a Veículos Brasil. Nesse caso utilizamos o modelo cloud, pois seria mais prático, com facilidade de manutenção, suporte 24h, custo fixo, escalabilidade, e flexibilidade.
